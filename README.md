# Milestone 01 - Employee REST API (Servlet)

## Overview
This milestone implements a basic RESTful API using Java Servlets and Apache Tomcat.  
It provides CRUD operations for managing employees and is tested using Postman.

## Technologies
- Java (JDK)
- Servlet / J2EE
- Apache Tomcat
- RESTful API Principles
- Postman

## How to Run
1. Open the project in IntelliJ / Eclipse
2. run Tomcat Server
